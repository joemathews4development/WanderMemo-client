## 📖 Description

WanderMemo is a modern travel memory and trip planning application that allows users to document their journeys, share travel experiences, and discover memories shared by fellow travelers.

Users can create trips, upload travel memories with photos, interact with other travelers through reactions and comments, and visualize journeys on interactive maps. WanderMemo also integrates an AI-powered trip planner that generates personalized itineraries based on user preferences.

The application focuses on a clean UI, responsive design, and a social travel experience.

## ✨ Features

### 🌍 Travel Memories
- Create and manage travel trips
- Upload travel memories with photos
- Add captions, cost, and activity types
- View memories in a timeline and grid view

### 📸 Media Upload
- Upload high-quality travel images using Cloudinary
- Manage and preview uploaded media

### 🗺 Interactive Maps
- Visualize travel routes and destinations using React Leaflet
- Display cities visited in a trip

### 🤝 Social Travel Experience
- Follow other travelers
- View feeds from travelers you follow
- React to memories with emojis
- Comment on memories

### 🤖 AI Trip Planner
- Generate travel itineraries using AI
- Select cities, number of days, and travel preferences
- AI generates a daily itinerary with activities

### ⭐ Premium Membership
- Upgrade to premium using Stripe
- Search for other WanderMemo travelers
- Follow and discover travel stories

### 🎨 Modern UI
- Material UI based design
- Fully responsive layout
- Light / Dark mode support

### ⚠️ Error Handling
- Custom 404 page
- Loading states for API calls
- Form validations

## User Stories

- **Signup / Login** – As a user, I want to create an account and log in so that I can access my travel memories.
- **Trips** – As a user, I want to create trips so that I can organize my journeys.
- **Memories** – As a user, I want to upload travel memories with photos and captions.
- **Feed** – As a user, I want to see memories shared by travelers I follow.
- **Reactions** – As a user, I want to react to memories using emojis.
- **Comments** – As a user, I want to comment on travel memories.
- **Maps** – As a user, I want to visualize my trips on a map.
- **AI Trip Planner** – As a user, I want AI to generate travel itineraries based on my preferences.
- **Premium Membership** – As a user, I want to upgrade to premium so that I can discover and follow other travelers.
- **Responsive UI** – As a user, I want the application to work smoothly on mobile and desktop devices.

## 📍 Routes

| Path | Page | Description |
|-----|-----|-------------|
| `/` | HomePage | User dashboard with trips and follow requests |
| `/feeds` | FeedsPage | Travel memory feed |
| `/users` | UsersPage | Search and follow travelers (premium feature) |
| `/account` | AccountsPage | Manage profile and subscription |
| `/about` | AboutPage | Information about the application |
| `/login` | LoginPage | User login |
| `/signup` | SignupPage | User registration |
| `/ai-trip-planner` | AITripPlannerPage | AI-powered trip planner |
| `/trips/:tripId` | TripDetailsPage | Detailed view of a trip |
| `/payment-success` | PaymentSuccess | Detailed view of a payment success |
| `*` | ErrorPage | Not Found route |
